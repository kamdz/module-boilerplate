# module-boilerplate

> Module Boilerplate.

## Install

```
$ npm install @kamdz/module-boilerplate
```

## Usage

```js
const add = require("@kamdz/module-boilerplate");
// or
import add from "@kamdz/module-boilerplate";

add(2, 2);
//=> 4
```

## add(a, b)

### params

#### a

Type: `number`

#### b

Type: `number`

### return

Type: `number`\
Default: `0`

Result of adding a to b.
